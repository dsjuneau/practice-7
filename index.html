What method is used to add one item to the end of an array?

arry.push("element") adds an element to the end of the array, modifies the original array and returns the new length of the array.

What method is used to remove one item from the end of the array?

arry.pop(); removes the last element of the array, modifies the original array and returns the element that was removed

What methods do the same two things but from the beginning of the array?

arry.shift(); arry.unshift("element")

What method creates a new array with each element modified according to some function?

arry.map(e=>e*e); Returns a new array with each element squared. Does not modify the original array.

What methods creates a new array consisting of a subset of the original array?

arry.filter(function); arry.slice(start,end); Both return new arrays without modifying the original array.
